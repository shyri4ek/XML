XML
 1. Создать внешний репозиторий c названием XML.        /Repositories/New/Create a new repository/Repository name/XML/*Public/Add a README file/Create repository
 2. Клонировать репозиторий XML на локальный компьютер. $ git clone https://github.com/shyri4ek/XML
 3. Внутри локального XML создать файл “new.xml”.       $ touch new.xml 
 4. Добавить файл под гит.                              $ git status $ git add new.xml
 5. Закоммитить файл.                                   $ git commit -m "add new.xml file"
 6. Отправить файл на внешний GitHub репозиторий.       $ git push
 7. Отредактировать содержание файла “new.xml”          $ vim new.xml/ i / вводим текст
 - написать информацию о себе                           <информация>
 (ФИО, возраст, количество домашних животных,           <ФИО>Попова Александра Владимировна</ФИО>
 будущая желаемая зарплата).                            <возраст>37</возраст>
 Всё написать в формате XML.                            <количество домашних животных>1</количество домашних животных>
                                                        <зарплата>2132 евро в месяц</зарплата>
                                                        </информация>                            
                                                        / Esc /:wq
 8. Отправить изменения на внешний репозиторий.  $ git add new.xml 
                                                 $ git commit -m "add text to new.xml" 
                                                 $ git push
 9. Создать файл preferences.xml                 $ touch preferences.xml

 10. В файл preferences.xml добавить информацию  $ vim preferences.xml / i / вводим текст
 о своих предпочтениях (Любимый фильм,                 <информация о предпочтениях>
 любимый сериал, любимая еда, любимое время года,      <Любимый фильм>Дневник памяти</Любимый фильм>
 страна которую хотели бы посетить) в формате XML.     <Любимый сериал>Секс в большом городе</Любимый сериал>
                                                       <Любимая еда>Фрукты/овощи</Любимая еда>
                                                       <Любимое время года>Лето</Любимое время года>
                                                       <Любимая страна>Индия</Любимая страна>
                                                       </информация о предпочтениях>
                                                       / Esc /:wq

 12. Сделать коммит в одну строку.    $ git add . 
                                      $ git commit -m "add text to preferences.xml and skills.xml"
 

 13. Отправить сразу 2 файла на внешний репозиторий.   $ git push
 14. На веб интерфейсе создать файл bug_report.xml.    / Add file / Create new file / Name your file... / bug_report.xml /Commit new file

 15. Сделать Commit changes (сохранить) изменения на веб интерфейсе. /в строке Commit changes пишем новый Commit    

 16. На веб интерфейсе модифицировать файл bug_report.xml,           
 добавить баг репорт в формате XML.
/ bug_report.xml / Edit file / вводим текст
<Название>Не работает кнопка отправки заказа.</Название>
<Действия>Открыть сайт *. Нажать по ссылке *. Пролистать до поля *. Ввести значение в поле *. Попытаться нажать кнопку отправки заказа.</Действия>
<Фактический результат>Кнопка неактивна.</Фактический результат>
<Ожидаемый результат>Кнопка нажимается, можно сделать заказ.</Ожидаемый результат>
<Вложения>Скрины, файлы, видео.</Вложения>

 17. Сделать Commit changes (сохранить) изменения на веб интерфейсе. /в строке Commit changes пишем новый Commit    

 18. Синхронизировать внешний и локальный репозиторий XML.           $ git pull
